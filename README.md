# StayFit - веб-додаток для контролю харчування

**StayFit** - це веб-додаток для підрахунку калорій та макронутрієнтів (білки, жири, вуглеводи).  
Мета додатку - допомогти користувачам планувати харчування, контролювати калорії та досягати своїх цілей у підтримці здорового способу життя.

---

## Основні функції

- **Реєстрація та збереження профілю користувача**  
  Створення акаунту та збереження персональних даних.
- **Додавання продуктів та страв**  
  Введення продуктів вручну або обрання зі списку.
- **Автоматичний підрахунок калорій і макронутрієнтів**  
  Білки, жири, вуглеводи для кожного дня.
- **Щоденні підсумки та історія харчування**  
  Перегляд спожитих калорій та БЖВ.
- **Дошка аналітики**  
  Графіки та діаграми споживання калорій та БЖВ по днях/тижнях.
- **Встановлення таргетів**  
  Налаштування щоденної норми калорій та пропорцій БЖВ.
- **Простий та інтуїтивний інтерфейс**  
  Швидкий доступ до головних функцій.

---

## Мета проекту

1. Створити інструмент для щоденного контролю харчування.
2. Легко підраховувати калорії та макронутрієнти.
3. Надати візуальну аналітику і прогрес у досягненні цілей.
4. Забезпечити збереження даних та історії споживання продуктів.
